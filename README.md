# projeto-Landing-Page
Projeto final do módulo base de Tecnologia da DNC. Uma aplicação web para uma landing page, utilizando HTML, CSS e JavaScript.


## 🎯 Objetivo do Projeto

O projeto foi criado para consolidar o aprendizado nas tecnologias introdutórias da trilha de tecnologia, focando em boas práticas de estruturação de código, estilização e manipulação de elementos da página.

## ✨ Funcionalidades Principais

* **Funcionalidade 1:** Criação de uma interface de usuário responsiva.
* **Funcionalidade 2:** Formulário de contato com validação de campos.

## 💻 Tecnologias Utilizadas

Este projeto foi desenvolvido com as seguintes tecnologias:

* **Frontend:**
    * HTML5
    * CSS3
    * JavaScript
* **Ferramentas:**
    * Git e GitHub
    * VS Code
